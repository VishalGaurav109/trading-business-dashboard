# 📊 Trading Business Performance Dashboard

## 🚀 Overview
This project analyzes trading business performance using SQL and Power BI. It provides insights into client activity, trading volume, revenue, and customer engagement.

---

## 📊 Key Metrics
- Total Clients: 2000  
- Active Clients: 1710  
- Active Rate: 86%  
- Total Trading Volume: ₹2.5 Billion  
- Total Revenue: ₹2.5 Million  

---

## 🧱 Data Model
The project follows a star schema:

- Clients (dimension table)  
- Trades (fact table)  
- Accounts (status tracking)  

---

## 📊 Dashboard Features
- KPI Cards (Clients, Revenue, Trading Volume, Active Users)
- Monthly Trading Trend
- Segment-wise Distribution
- City-wise Performance Map
- Top Clients Analysis
- Support Ticket Insights

---

## 🛠 Tools Used
- SQL (MySQL)
- Power BI
- DAX

---

## 💡 Key Insights
- ~86% clients are actively trading  
- Equity segment dominates trading volume  
- Revenue is derived using brokerage logic
